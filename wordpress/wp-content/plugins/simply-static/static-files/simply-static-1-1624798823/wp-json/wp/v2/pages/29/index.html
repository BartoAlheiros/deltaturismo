A vila \u00e9 um dos melhores destinos de praia n\u00e3o apenas do Nordeste, mas do Brasil. N\u00e3o \u00e9 preciso dizer muito para explicar o lugar: suas praias, caracterizadas pelas \u00e1guas verdes e pelas piscinas naturais, saltam aos olhos e arrancam elogios dos mais descrentes.<\/p>\t\t\t\t\t\t<\/div>\n\t\t\t\t<\/div>\n\t\t\t\t
